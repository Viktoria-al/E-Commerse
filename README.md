Анализ продукта: E-commerce воронка продаж

## Описание проекта

Учебный проект по продуктовой аналитике.

Цель проекта — проанализировать поведение пользователей интернет-магазина,
найти точки потери пользователей в воронке и определить возможности роста продукта.

## Используемые инструменты

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL

## Что было сделано

- Проверка качества данных
- Расчёт ключевых продуктовых метрик
- Анализ конверсионной воронки
- Анализ эффективности каналов привлечения
- Анализ выручки
- Анализ влияния скидок
- Сегментация пользователей
- Анализ устройств
- Когортный анализ удержания пользователей.

## Основные выводы

- Основная потеря пользователей происходит между просмотром товара и добавлением в корзину.
- Email показывает высокую эффективность по выручке на пользователя.
- Paid Ads приводит большой объём пользователей, но требует оптимизации качества трафика.
- Удержание пользователей снижается после первого взаимодействия.
